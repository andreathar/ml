LIMITATIONS & CONSTRAINTS
----------------------------------------------------------------------

  2. Transform interpolation handled by Motion system, not direct Transform manipulation
  3. Mannequin is the visual representation - separate from Character logic

  5. Animation states managed through State assets, not direct Animator control
  6. Input system uses InputValue wrappers - never raw Input class
  7. IK targets must be set through Character.IK system, not direct bone manipulation
  8. Skeleton building requires proper bone mapping in Mannequin settings
  9. State transitions use blending - abrupt changes cause visual artifacts
  10. Driver system (NavMesh/CharacterController) determines movement authority
