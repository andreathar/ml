# Unity ML Project

This is a Unity project configured for machine learning capabilities and agent development.

## Project Setup

### Prerequisites
- Unity 2022 LTS or later
- .NET SDK (for building)
- Git

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd ml
```

2. Open the project in Unity Hub and select the project folder

3. Ensure all dependencies are installed via the Package Manager

## Project Structure

- **Assets/** - Contains all Unity game assets and scripts
- **Packages/** - Unity package dependencies
- **ProjectSettings/** - Unity project configuration
- **Library/** - Generated by Unity (not tracked in git)
- **Logs/** - Build and runtime logs
