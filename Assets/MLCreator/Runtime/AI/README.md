# MLCreator AI

Artificial intelligence and NPC behavior systems.

## Purpose

This module provides AI functionality that integrates with GameCreator Perception:
- Behavior state machines
- Decision-making systems
- Custom sensors and detection

## Subdirectories

- `Behaviors/` - AI state machines and behavior logic
- `Sensors/` - Custom perception sensors
- `Decisions/` - Utility-based decision systems

## Namespace

`MLCreator.Runtime.AI`

## Integration

Works with:
- `GameCreator.Runtime.Perception` - Sensing systems
- `GameCreator.Runtime.Characters` - Character control
- `GameCreator.Runtime.Behavior` - Behavior trees (if installed)

## Usage

Use `/mlcreator-ai` Claude skill for guidance on creating AI systems.
