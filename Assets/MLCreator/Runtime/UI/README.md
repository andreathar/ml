# MLCreator UI

User interface components and systems.

## Purpose

This module provides UI functionality:
- HUD elements (health bars, indicators)
- Menu systems
- UI components that integrate with GameCreator

## Subdirectories

- `HUD/` - Heads-up display elements
- `Menus/` - Menu controllers and panels
- `Components/` - Reusable UI components

## Namespace

`MLCreator.Runtime.UI`

## Integration

Works with:
- `UnityEngine.UI` - Core UI system
- `TMPro` - TextMeshPro text
- `GameCreator.Runtime.Stats` - Stat display
- `GameCreator.Runtime.Common.UnityUI` - GC UI utilities

## Usage

Use `/mlcreator-ui` Claude skill for guidance on creating UI components.
