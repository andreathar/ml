# MLCreator Core

Core systems, managers, and initialization logic for MLCreator.

## Purpose

This module provides foundational systems that other modules depend on:
- Singleton managers for global access
- Initialization and bootstrap code
- Shared utilities and helpers

## Subdirectories

- `Managers/` - Singleton manager classes (e.g., `MLCreatorManager`)
- `Initialization/` - Bootstrap and startup code
- `Utilities/` - Shared utility classes and helpers

## Namespace

`MLCreator.Runtime.Core`

## Usage

Use `/mlcreator-core` Claude skill for guidance on creating core systems.
